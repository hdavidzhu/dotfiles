# dotfiles

### Setup

```zsh
# First, install zsh
# https://github.com/ohmyzsh/ohmyzsh/wiki/Installing-ZSH#install-and-set-up-zsh-as-default

# Install Antigen, the zsh package manager
> touch ./dotfiles/libs/antigen.zsh
> curl -L git.io/antigen > ./dotfiles/libs/antigen.zsh

# Run the general install script
> ./dotfiles/install.sh

# To refresh
> ss
```
